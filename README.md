# My Terminal Config

## ZSH
### Ubuntu:
([Source](https://www.howtoforge.com/tutorial/how-to-setup-zsh-and-oh-my-zsh-on-linux/))  
Install:   
`apt install zsh`

Set default shell:  
`chsh -s /usr/bin/zsh root`

Check default shell:  
`echo $SHELL`


## Oh My Zsh
([Source](https://github.com/robbyrussell/oh-my-zsh))

Need `wget` and `git`  
Install:  
`sh -c "$(wget -O- https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"`
